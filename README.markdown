Folder Git
==========

[![Build Status](https://travis-ci.org/l0b0/fgit.svg)](https://travis-ci.org/l0b0/fgit)

Run a Git command in several repositories.

Test
----

Requires `shunit2`.

    git submodule update --init
    make test

Install
-------

    sudo make install

`make test` depends on `shunit2`.

Usage
-----

    fgit --help
